## 🎨 CUSTOMIZATION GUIDE

### Change Contact Information
Edit these files:
1. **src/components/Navigation.js** - Update header contact
2. **src/components/Footer.js** - Update footer contact
3. **src/pages/Contact.js** - Update contact page details

Example:
```javascript
// Find and replace:
justica@gmail.com → youremail@example.com
(+1)-987654321 → (+1)-YOUR-NUMBER
Tobermory, ON, Canada → Your Address
```

### Update Team Members
Edit **src/pages/Home.js** and **src/pages/Team.js**

Replace team member objects:
```javascript
{
  name: 'Sophia Martinez',
  title: 'Senior Associate, Employment Law Division',
  image: 'https://framerusercontent.com/images/...'
}
```

### Change Colors
Edit **src/styles/App.css**

Main colors:
- `#1a3a52` - Dark blue (primary)
- `#d4af37` - Gold (accent)
- `#f5f5f5` - Light gray (background)

### Replace Images
All images use direct URLs. To use your own:

**Option 1: Direct URL**
```javascript
image: 'https://your-website.com/image.jpg'
```

**Option 2: Local Image**
1. Save images in `public/images/`
2. Reference them:
```javascript
image: './images/your-image.jpg'
```

## 🐛 TROUBLESHOOTING

### Port 3000 Already in Use
```bash
PORT=3001 npm start
```
App will run on port 3001 instead

### Dependencies Not Installing
```bash
rm -rf node_modules package-lock.json
npm install
```

### Module Not Found Error
```bash
npm install react-router-dom
```

### Styles Not Loading
Clear browser cache (Ctrl+Shift+Delete) and refresh
