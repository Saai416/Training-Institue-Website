## 🎯 QUICK CUSTOMIZATION

### Change Contact Info (5 min)
Edit these files:
1. `src/components/Navigation.js` → Line 10-15
2. `src/components/Footer.js` → Line 15-30
3. `src/pages/Contact.js` → Line 50-80

Find and replace:
- `justica@gmail.com` → your email
- `(+1)-987654321` → your phone
- `Tobermory, ON, Canada` → your address

### Update Team Members (10 min)
Edit `src/pages/Home.js` → Line 40-70
Replace each team member object:
```javascript
{
  name: 'Your Name',
  title: 'Your Title',
  image: 'https://your-image-url.jpg'
}
```

### Change Colors (5 min)
Edit `src/styles/App.css` → Lines 30-40
- Primary: `#1a3a52` (dark blue)
- Accent: `#d4af37` (gold)
- Background: `#f5f5f5` (light gray)

### Replace Images (5 min)
Find image URLs in components and replace with your own:
```javascript
// Before:
image: 'https://framerusercontent.com/images/...'

// After:
image: 'https://your-website.com/image.jpg'
```
